# Todoist Terminal Client Usage Guide (HOWTO)

이 프로그램은 키보드 기반(Keyboard-driven)으로 Todoist 작업을 관리할 수 있도록 설계되었습니다. 아래의 단축키를 활용하여 터미널에서 빠르게 작업을 처리해 보세요.

## 🌐 글로벌 단축키 (Global Shortcuts)
어느 창에서든 사용할 수 있는 기본 단축키와 필터링/정렬 기능입니다.

- `h`: 도움말(Help) 창 열기 / 닫기
- `q`: 프로그램 종료
- `Tab`: 프로젝트(Projects) 목록과 작업(Tasks) 목록 간 포커스 이동
- **작업 필터 및 정렬:**
  - `t`: 오늘(Today) 해야 할 작업 보기
  - `o`: 기한이 지난(Overdue) 작업 보기
  - `a`: 모든(All) 작업 보기
  - `p`: 우선순위(Priority) 기준으로 정렬
  - `d`: 날짜(Date) 기준으로 정렬

---

## 📝 작업 목록 (Tasks Pane)
작업 목록 창에 포커스가 있을 때 사용하는 단축키입니다.

- `j` 또는 `↓`: 아래(다음) 작업으로 이동
- `k` 또는 `↑`: 위(이전) 작업으로 이동
- `Enter`: 선택된 작업의 편집기(Task Editor) 열기
- `x`: 선택된 작업 완료(Close) 처리
- `d`: 선택된 작업 영구 삭제(Delete)
- `n`: 현재 선택된 프로젝트에 새로운 작업 추가

---

## 📁 프로젝트 목록 (Projects Pane)
프로젝트 목록 창에 포커스가 있을 때 사용하는 단축키입니다.

- `j` 또는 `↓`: 아래(다음) 프로젝트로 이동
- `k` 또는 `↑`: 위(이전) 프로젝트로 이동
- `+` 또는 `n`: 현재 선택된 프로젝트에 새로운 작업(Task) 추가

---

## ✏️ 작업 편집기 (Task Editor)
작업을 편집하거나 하위 작업을 관리할 때 사용합니다.

- `Tab`: 입력 필드 간 이동 (내용 -> 설명 -> 기한(Due) -> 하위 작업 순)
- `Shift + Tab`: 이전 입력 필드로 이동 (역순 이동) (@syzseisus)
- `Alt + Enter` 또는 `Ctrl + n`: Description 필드에서 줄바꿈
- `Esc`: 변경사항을 취소하고 편집기 닫기
- `Enter`: 수정한 내용을 서버에 저장하고 닫기 (하위 작업 리스트가 선택된 상태라면 해당 하위 작업을 엶)
- `n` (하위 작업 리스트 포커스 시): 현재 작업에 속하는 새로운 하위 작업(Sub-task) 생성
- 하위 작업 리스트에서 `j`, `k`, `↓`, `↑` 키를 통해 항목 이동이 가능합니다.

---

## ➕ 새 작업 추가 (New Task Prompt)
새로운 작업을 추가하는 창에서 사용합니다.

- `Tab`: 입력 필드 간 이동 (내용 -> 설명 -> 기한 순)
- `Shift + Tab`: 이전 입력 필드로 이동 (역순 이동) (@Syzseisus)
- `Alt + Enter` 또는 `Ctrl + n`: Description 필드에서 줄바꿈 (@Syzseisus)
- `Esc`: 작업 생성을 취소하고 창 닫기
- `Enter`: 입력한 내용으로 새 작업을 서버에 생성
